A simple Geometry Wars clone written in C++ using the SDL API.

Dependencies:
    - SDL 1.2
    - SDL_image
    - SDL_ttf
    - SDL_mixer
    - MSVC++ Redistributable 2010 x86

Controls ([D] - Debug mode only):
    Move                :   WASD / Arrow Keys
    Aim                 :   Mouse
    Fire                :   Left Mouse Button
    Debug Mode          :   ~
    Spawn Enemies [D]   :   Q
    Force EMP [D]       :   E
